习题 2：二分法求根

方程：f(x) = x³ + 4x² - 10 = 0　　区间：[1, 2]　　误差要求：≤ 10⁻³

Python 实现代码

a = 1.0
b = 2.0
误差要求 = 0.001
次数 = 0

print("初始区间:", a, b)

while b - a > 误差要求:
    c = (a + b) / 2
    fc = c**3 + 4*c**2 - 10
    
    print(f"第{次数:2d}次  中点:{c:.6f}  f(c)={fc:.6f}  区间长:{b-a:.6f}")
    
    if fc == 0:
        break
    elif (c**3 + 4*c**2 - 10) * (a**3 + 4*a**2 - 10) < 0:
        b = c
    else:
        a = c
        
    次数 += 1

print("\n近似根 ≈", (a + b)/2)
print("二分次数:", 次数)
print("最后区间长度:", b - a)
